# Student Feedback Survey Form

A clean, responsive, and accessible **Student Feedback Survey Form** built using **HTML5 and CSS3**.  
This project focuses on modern form design, usability, and theming, and is intended for learning as well as portfolio showcase.

---

## Project Overview

The Student Feedback Survey Form allows users to share their learning experience through a structured and user-friendly interface.  
It demonstrates best practices in semantic HTML, modern CSS, and responsive design — without using JavaScript.

---

## Features

- **Fully Responsive** (Mobile-first design)
- **Dark & Light Theme Support** (based on system preference)
- **Accessible Form Structure**
  - Semantic HTML (`fieldset`, `legend`, `label`)
  - Keyboard-friendly focus states
- **Clean UI with Modern CSS**
  - CSS Variables
  - Smooth hover & focus interactions
- **Well-structured & Maintainable Code**
- No JavaScript — pure HTML & CSS

---

## Technologies Used

- **HTML5**
- **CSS3**
  - Flexbox
  - CSS Custom Properties (Variables)
  - Media Queries
  - `prefers-color-scheme`
